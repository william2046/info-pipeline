    *   **发生了什么**: ChatGPT 的默认模型已更新为 GPT-5.5 Instant，提供更智能、更准确的回答，减少幻觉，并改进了个性化控制。

3.  **ChatGPT Images 2.0 发布**
    *   **发生了什么**: ChatGPT Images 2.0 引入了最先进的图像生成模型，显著提升了文本渲染、多语言支持和高级视觉推理能力。
    *   **为什么重要/对读者有什么用**: 用户现在可以通过 ChatGPT 生成更高质量、更精准的图像，尤其在处理文本和复杂概念时，极大地拓展了创意和设计可能性。
    *   **链接**: https://openai.com/index/introducing-chatgpt-images-2-0
4.  **OpenAI API 推出全新实时语音模型**
    *   **发生了什么**: OpenAI API 中新增了可以进行推理、翻译和转录语音的实时语音模型，支持更自然、更智能的语音体验。
    *   **为什么重要/对读者有什么用**: 开发者可以利用这些模型构建低延迟、高智能的语音应用，为客户服务、教育和无障碍技术等领域带来革命性变革。
    *   **链接**: https://openai.com/index/advancing-voice-intelligence-with-new-models-in-the-api
5.  **Codex 应用扩展至 macOS 和 Windows**
    *   **发生了什么**: 更新后的 Codex 应用现已支持 macOS 和 Windows，增加了计算机使用、应用内浏览、图像生成、记忆和插件功能，以加速开发者工作流。
    *   **为什么重要/对读者有什么用**: 这使得 Codex 成为一个更全面的桌面级开发工具，开发者可以在本地环境中更高效地自动化任务、连接工具并生成实际输出。
